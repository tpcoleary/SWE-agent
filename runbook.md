## ūüß† Memory Integration (Mem0)

SWE-agent connects to the shared local Mem0 instance (`http://localhost:8888`) via `mem0_client.py`. Use `run_with_memory.sh` as a drop-in replacement for `sweagent run` ‚ÄĒ it recalls relevant context before the run and records the outcome after.

| File | Purpose |
|---|---|
| `~/swe-agent/mem0_client.py` | Zero-dependency Mem0 REST client |
| `~/swe-agent/mem0_session.py` | Pre/post-run hook CLI (`search`, `record`) |
| `~/swe-agent/run_with_memory.sh` | Drop-in wrapper for `sweagent run` with memory |

**Usage:**
```bash
./run_with_memory.sh --agent.model.name="openai/nemotron3super_autogen" \
  --env.repo.path="/home/tpcoleary/myrepo" \
  --problem_statement.path="/home/tpcoleary/myrepo/issue.md"
```

| Item | Value |
|---|---|
| **agent_id** | `swe-agent` |
| **user_id** | `tpcoleary` |
| **Backend** | `http://localhost:8888` (local Docker) |
